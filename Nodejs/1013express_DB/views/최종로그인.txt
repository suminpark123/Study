메인으로 돌아가도 로그인 상태를 유지할 수 있는
최종 로그인 / 로그아웃 기능을 만들어보자!
1)  로그인 성공 시                   => Session에 저장
    LoginS.html 페이지에
    '000님, 로그인에 성공하셨습니다'  => EJS 로 변경
    '메인으로 돌아가기' => 버튼
2) 메인페이지
    만약, 로그인을 했다면
    현재 로그인 중인 아이디 : xxx      => EJS 변경
    로그인 버튼 => 로그아웃 버튼
3) 로그아웃 버튼 클릭 시, 세션이 사라지도록     => Session 삭제
    => 메인페이지로 이동                      => 메인EJS 로 이동
        현재 로그인 어쩌고 텍스트 안뜨게 하기
        로그아웃 버튼 => 로그인 버튼
